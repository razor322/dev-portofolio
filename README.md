A minimal dev portfolio showcasing my work as a Software Engineer.

→ Check out the live site: [gybrannauval.site](https://gybrannauval.site)

## Overview

### Stack

- Next.js 16
- Tailwind CSS v4
- shadcn/ui
- TypeScript
- MDX

### Featured

- Clean & modern design
- Light/Dark themes
- vCard integration
- SEO optimized (JSON-LD schema, sitemap, robots)
- AI-ready with /llms.txt
- Spam-protected email
- Installable as PWA

### Content

Centralized document system powered by MDX:

- Category-based content organization
- Syntax highlighting with code blocks
- RSS feed for content distribution

## Development

```bash
pnpm dev             # Dev server
pnpm build           # Production build
pnpm lint            # ESLint
pnpm format:write    # Prettier
pnpm check-types     # Type checking
```
